Quid2

Evaluate expressions in Quid2 (pronounced as 'quidquid'), a distributed computing system.

Tip: Press Tab or Return to accept suggestion.

Some examples (click on the expression to evaluate it):

3.3+3.7
Simple arithmetic.
[1,2,3] or 11 : [22,33]
Lists.
Data.List.take 11 $ Math.fibonacci
List operations: take the first eleven elements of an infinite list of integers.
Money.value $ Finance.ExchangeRate Fact.Currency.GBP Fact.Currency.EUR
IO operations: find the current value of the GBP/EUR exchange rate.
env
And to see how it all works: show the environment where expressions are evaluated.
